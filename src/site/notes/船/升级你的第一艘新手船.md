新手运力不足,寻求运力提升,常见疑惑大致如下:

- 运力怎么提升?
- 哪种升级船型更适合我?(LCB 和 WCB 的区别)
- 怎么把新手船升级成心仪船型?(蓝图、船厂介绍)
- 得到配件后去哪升级?

---

## 运力怎么提升?

两种方式:**升级现有船只** 或 **购买全新船只**。

对于新手 **只推荐升级**:

| 同容量飞船 | 花费    |
| ----- | ----- |
| 升级    | 100w+ |
| 购买    | 500w+ |

目前市面上提供两种升级方向:

- 升级为 **LCB**(2K / 2K)
- 升级为 **WCB**(3K / 1K)

---

## 哪种升级船型更适合我?

### 两种船型对比

| 船型 | 容量(重量/体积) | 价格 |
|---|---|---|
| **LCB** | 2000t / 2000m³ | 227w |
| **WCB** | 3000t / 1000m³ | 93w |

> 价格来源:GTU 造船厂

### 选择建议

<details>
<summary>从货物类型看</summary>
<ul>
<li>运输农产、水等<b>重量与体积接近</b>的物品 → 推荐 LCB</li>
<li>运输矿石、金属等<b>大重量</b>的物品 → 推荐 WCB</li>
</ul>
</details>

<details>
<summary>从性价比、长期看</summary>
<ul>
<li>WCB 价格远低于 LCB</li>
<li>长期游玩多半会涉及多个领域,或多或少都会购买 WCB</li>
</ul>
</details>

<details>
<summary>从泛用性看</summary>
<ul>
<li>有时建立新基地建材体积会超过 1K,使用 WCB 要运两趟</li>
<li>LCB 更适合倒货(可用更便宜的 STL-LCB 代替)</li>
</ul>
</details>

## 怎么把新手船升级成心仪船型?

### 第一步:用指令打开蓝图
```
BLU
```

**蓝图** 是定义船只的地方,可以根据个人需求选择不同配件 —— 比如更快的船、更大空间的船、损耗更低的船。选好配件后,蓝图会列出制造该船所需的全部材料(配件)。

[制造新手船所需材料](Pasted%20image%2020260620225227.png)

新手初始会有两张蓝图,只关注 **BP-STRT-0000**,这是你当前新手船的蓝图。

### 升级逻辑

> 创建目标飞船蓝图 → 得到所需配件 → 补足/更换缺少的配件

[市面上常见两种固定蓝图](Pasted%20image%2020260620230631.png)

[三种船型的配件差距](Pasted%20image%2020260620230842.png)

### 第二步:购买配件

按照图片创建蓝图,然后去船厂(玩家扮演的一种身份)购买对应配件。

**注意**:
- 船厂按 **升级包** 出售,而非单卖
- **不推荐用 CX 购买**,价格会贵很多
- 通常需等待几天,可根据计划提前订购

[升级包示例](Pasted%20image%2020260620231300.png)

> 图片来源:GTU 造船厂

## 得到配件后去哪升级?

去船厂(游戏中真实存在的一种建筑)升级。

### 找到船厂位置

用指令查看星系船厂分布:
```
MS 星系名
```

[例如](Pasted%20image%2020260620231913.png)
```
MS VH-331
```

右上角点亮"船厂"图标,带图标的星球即存在船厂。**通常选择离空间站最近的星球** —— 例子中 Bernironia 星球最近,也就是 **VH-331c**。

### 准备工作

通常卖船给你的玩家会通过合同把配件放在你的空间站。你需要:

- 把所有配件、**以及足够返回空间站的燃料** 一并运往拥有船厂的星球
- **被升级的船只必须在该星球**

### 升级流程

打开星球界面后,按以下顺序操作:

1. 点击 **船厂**
2. 填入 **初始蓝图** 与 **目标蓝图**
3. **创建项目**
4. **查看项目** → 选择待升级船只
5. **填满材料** → **分配** → **开始**
6. 等待完成
